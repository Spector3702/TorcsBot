# Quick Start

## Set Python path
* Linux or MACOS
    ```shell=
    export PYTHONPATH="${PYTHONPATH}:path/to/SelfDrivingCar"
    ```

## Run DDPG
* Train
    ```python=
    python TORCS_DDPG/train.py
    ```

## Run NEAT
* Train
    ```python=
    python TORCS_NEAT/train.py
    ```
